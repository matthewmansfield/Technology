== Description ==

Provides readers a way to copy/paste citation of your articles, pages, or blog posts. Insert the citation section using the shortcode "[wpcitation]". Usually, citations are placed at the bottom-most portion of any article; however, you do you.

= Main Features =

*	Very easy to install 
*	Automatically creates a citation where ever the shortcode is inserted 
*   Works in all major browsers - IE7, IE8, IE9, Safari, Firefox, Chrome
*	Add to any page using shortcode "[wpcitation]"

== Installation ==

1. Upload the 'WP-Citation' folder to the '/wp-content/plugins/' directory
1. Activate the WP Citation plugin through the WordPress 'Plugins' dashboard.
1. Configure the plugin by going to the `WP Citation` tab that appears in your admin menu.
1. Add to any page using shortcode `[wpcitation]`
